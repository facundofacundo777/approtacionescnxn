Falta conectar Firebase

La app necesita un proyecto de Firebase para guardar los datos y manejar el acceso.

  1. Entrá a console.firebase.google.com y creá un proyecto.
  2. Activá Authentication → Google y Firestore Database.
  3. En Configuración del proyecto, copiá el objeto firebaseConfig.
  4. Abrí este archivo HTML y pegalo donde dice PEGAR_CONFIG_ACA.
  5. Agregá tu email en ADMIN_EMAILS para entrar como administrador.
